# FreshSense AI 🥬🤖

**AI-Powered Food Freshness Analysis with Recipe Suggestions**

Transform your smartphone into a food safety expert using advanced AI technology. FreshSense analyzes food images to assess freshness, provides safety recommendations, and generates personalized recipes from your ingredients.

![FreshSense AI Banner](https://images.unsplash.com/photo-1556909114-f6e7ad7d3136?w=800&h=300&fit=crop&crop=center)

## ✨ Features

### 🔍 **AI Food Analysis**
- **Instant Freshness Assessment**: Upload or capture food images for immediate AI analysis
- **Visual Quality Indicators**: Detailed assessment of color, texture, and spoilage signs
- **Safety Recommendations**: Expert-level guidance on food safety and consumption timing
- **Confidence Scoring**: AI confidence levels for informed decision-making

### 👨‍🍳 **Smart Recipe Generation**
- **Personalized Recipes**: AI-generated recipes based on your available ingredients
- **Difficulty Levels**: Recipes for all skill levels from beginner to expert
- **Nutritional Information**: Complete nutritional breakdown for health-conscious cooking
- **Cooking Tips**: Professional chef recommendations and preparation guidance

### 🛒 **Integrated Shopping**
- **One-Click Ingredient Shopping**: Direct links to purchase missing ingredients
- **Price Optimization**: Find the best deals on fresh ingredients
- **Shopping Lists**: Automated shopping lists for recipe ingredients
- **Amazon Fresh Integration**: Seamless integration with Amazon for quick delivery

### 📱 **Mobile-First Experience**
- **Camera Integration**: Use your device camera for instant food capture
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Progressive Web App**: Install on your device for native app experience
- **Offline Capability**: Save recipes and analysis history for offline viewing

### 📊 **Personal Dashboard**
- **Analysis History**: Track all your food scans and assessments
- **Recipe Collection**: Save and organize your favorite AI-generated recipes
- **Purchase Tracking**: Monitor your ingredient purchases and spending
- **Activity Timeline**: View your food analysis and cooking journey

## 🚀 Quick Start

1. **Visit the App**: [FreshSense AI](https://freshsense-ai.vercel.app)
2. **Take a Photo**: Use your camera or upload an image of your food
3. **Get Analysis**: Receive instant AI-powered freshness assessment
4. **Discover Recipes**: Explore personalized recipe suggestions
5. **Start Cooking**: Follow step-by-step instructions with shopping integration

## 🤝 Contributing

FreshSense AI is built with modern web technologies and welcomes contributions from the community.

### 🛠️ Development Setup
```bash
# Clone the repository
git clone https://github.com/mdelaguera/freshsense-ai.git

# Install dependencies
npm install

# Start development server
npm run dev
```

### 🔧 Environment Variables
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
OPENAI_API_KEY=your_openai_key
```
